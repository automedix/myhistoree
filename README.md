# myhistoree

**Online Anamnese für den erstmaligen Praxisbesuch**

Digitale Anamneserhebung, die Patienten bequem von zuhause aus vor dem ersten Besuch ausfüllen können.

## Konzept

- **Vorbefragung**: Patienten tragen ihre Beschwerden, Vorerkrankungen und Medikamente online ein
- **Zeitersparnis**: Weniger Wartezeit und Fragebögen in der Praxis
- **Strukturierte Daten**: Ärzte erhalten übersichtliche, digitale Anamnese
- **Datenschutz**: Ende-zu-Ende-Verschlüsselung, DSGVO-konform

## Funktionen
- Symptom-Check mit Leitfaden
- Medikamenten- und Allergie-Liste
- Vorerkrankungen & Operationen
- Familienanamnese
- Upload von Befunden/Laboren

## Technologie
- Web-basiert (mobile-first)
- Verschlüsselte Datenübertragung
- NOSTR-Integration für dezentrale Identität (optional)
